# Task GH-004: Base Layout with Bootstrap 5.3 and Theme Switcher

## Objective
Create the base Blade layout with Bootstrap 5.3, header, footer, and dark/light/system theme toggle.

## Requirements

### 1. Base Layout Structure
- Create `resources/views/layouts/app.blade.php`
- Include Bootstrap 5.3 CSS and JS from CDN
- Responsive meta tags
- Clean, semantic HTML5 structure

### 2. Header Component
- Navigation bar using Bootstrap navbar
- Left side: Application name (get from `config('app.name')`)
- Right side: Theme toggle button (icon button)
- Responsive design (mobile-friendly)

### 3. Footer Component
- Fixed or static footer at bottom
- Text format: "{appname} {version} built with love by Devnodes.in"
- Get app name from `config('app.name')`
- Get version from `config('app.version')`
- Center-aligned text
- Subtle styling

### 4. Theme Switcher Functionality
- Three modes: Light, Dark, System (auto)
- Toggle button cycles through modes or dropdown menu
- Save selected theme to localStorage
- Apply theme on page load
- Use Bootstrap 5.3's built-in color modes (data-bs-theme attribute)
- JavaScript implementation for theme persistence

### 5. Main Content Area
- Yield section for page-specific content
- Proper spacing and padding
- Container or container-fluid depending on layout needs

## Acceptance Criteria
- [ ] Base layout renders correctly with Bootstrap 5.3
- [ ] Header displays app name and theme toggle
- [ ] Footer displays correct app name and version
- [ ] Theme switcher cycles through Light/Dark/System modes
- [ ] Selected theme is saved to localStorage
- [ ] Theme preference persists across page reloads
- [ ] Layout is responsive on mobile, tablet, and desktop
- [ ] All Bootstrap 5.3 components work correctly

## Technical Notes
- Use Bootstrap 5.3 color modes feature
- Store theme preference as 'theme' in localStorage
- Apply theme by setting data-bs-theme attribute on html or body element
- Use JavaScript to detect system theme preference
- Include icons for theme toggle (Bootstrap Icons or similar)
