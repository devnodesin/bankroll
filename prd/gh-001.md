# Task GH-001: Laravel Project Setup and Configuration

## Objective
Set up a new Laravel project with basic configuration for the Bankroll application.

## Requirements

### 1. Laravel Installation
- Create a new Laravel project (latest stable version)
- Install in the `src/` directory
- Configure environment variables in `.env`

### 2. Application Configuration
- Set application name to "Bankroll"
- Set application version in `config/app.php`
- Configure timezone appropriately
- Set up database connection (SQLite)

### 3. Dependencies
- Ensure Composer dependencies are installed
- Verify Laravel installation is working

### 4. Basic Structure
- Verify directory structure is correct
- Ensure all Laravel default directories exist
- Set proper file permissions

## Acceptance Criteria
- [ ] Laravel project successfully installed in `src/` directory
- [ ] Application name and version configured in `config/app.php`
- [ ] `.env` file properly configured
- [ ] Database connection tested and working
- [ ] Application runs successfully with `php artisan serve`

## Technical Notes
- Use latest stable Laravel version
- Follow Laravel best practices for directory structure
- Document any custom configuration in comments
